# 源步 GeneStep

从本源到步态，一站式足式机器人开源全栈平台。

From source to step, an all-in-one open-source full-stack platform for legged robots.


**源起足下，步向未来**

**Step with Source, Move with Full Stack**


GeneStep 源步是一套面向双足、四足足式机器人的全栈开源方案，涵盖机械结构、硬件电路、嵌入式驱动、运动控制算法与上位机软件，提供从图纸到代码、从原型到步态的完整开发体系。
以开放架构回归机器人运动本源，用全栈能力支撑每一步稳定行走，让足式机器人开发更简单、更自由。

GeneStep is a full-stack open-source solution for bipedal and quadrupedal robots, covering mechanical structure, hardware circuits, embedded drivers, motion control algorithms, and upper computer software. It provides a complete development system from design drawings to code, from prototype to stable gait.
With an open architecture, we return to the origin of robot motion, and empower every stable step with full-stack capabilities, making legged robot development simpler and more accessible.
